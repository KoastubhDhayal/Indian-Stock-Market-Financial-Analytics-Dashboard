🚀 QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

1️⃣  Extract the archive:
    tar -xzf indian-stock-analytics.tar.gz
    cd indian-stock-analytics

2️⃣  Install dependencies:
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt

3️⃣  Start the server:
    ./start.sh
    
    Or manually:
    python3 api_server.py

🌐 Access API at: http://localhost:5000

🎓 WHAT IS FINBERT?
═══════════════════════════════════════════════════════════════════════════════

FinBERT is a pre-trained NLP model specifically fine-tuned for financial 
sentiment analysis. It's based on BERT and achieves 94% accuracy on financial
texts.

Key Capabilities:
• Analyzes news headlines and articles
• Monitors social media sentiment (Twitter, Reddit)
• Classifies text as Positive/Negative/Neutral
• Provides confidence scores
• Detects sentiment momentum (improving/deteriorating)
• Generates trading signals from sentiment

Example:
Input:  "Company reports record profits, stock soars"
Output: POSITIVE (92% confidence)

Input:  "CEO resigns amid scandal, investors concerned"  
Output: NEGATIVE (88% confidence)

📊 EXAMPLE USAGE
═══════════════════════════════════════════════════════════════════════════════

Python:
    import requests
    
    response = requests.get('http://localhost:5000/api/stock/RELIANCE/complete')
    data = response.json()
    
    print(f"Price: ₹{data['data']['current']['price']}")
    print(f"Action: {data['data']['recommendation']['action']}")
    print(f"Sentiment: {data['data']['sentiment']['overall']['overall_sentiment']}")

cURL:
    curl "http://localhost:5000/api/stock/TCS/sentiment?days=7"

JavaScript:
    fetch('http://localhost:5000/api/stock/RELIANCE/recommendation')
      .then(res => res.json())
      .then(data => console.log(data));

🔧 CONFIGURATION
═══════════════════════════════════════════════════════════════════════════════

Edit config.py or .env file to customize:

• API_PORT: Server port (default 5000)
• DEFAULT_EXCHANGE: NSE or BSE
• CACHE_DURATION: Cache time in seconds
• PREDICTION_HORIZON: Days ahead to predict
• RSI_PERIOD: RSI calculation period
• ENABLE_ML_PREDICTIONS: Enable/disable ML
• Risk profiles: Conservative, Moderate, Aggressive

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

ModuleNotFoundError:
    pip install -r requirements.txt

FinBERT not loading:
    pip install transformers torch

No stock data:
    • Check internet connection
    • Verify stock symbol (add .NS for NSE)
    • Try: python3 stock_backend.py

Slow performance:
    • Enable Redis caching
    • Use GPU for FinBERT
    • Reduce fetch frequency
